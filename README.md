# Blender Zaku（ザク3Dシーン制作）

## 概要
Blenderを用いて、ザクの3Dモデルおよびシーン全体（背景・アニメーション・カメラ）を制作しました。  

## 使用技術
- Blender
- 3Dモデリング（ローポリゴン最適化）
- マテリアル設定
- アニメーション制作
- カメラワーク設定

## 制作内容
- ザク本体のモデリング
- 背景（ビル・地面・空）の構築
- アニメーション設定（動き・演出）
- カメラ視点の設計（映像としての見せ方）
- マテリアルの再現（既存デザインを参考）

## 工夫した点
- ポリゴン数を抑えた軽量モデルを設計中（ローポリゴン化）
- トポロジーを意識し、変形や編集しやすい構造を構築中
- カメラワークにより、モデルの見せ方を工夫

## 課題・改善点
- テクスチャのディテール強化
- よりリアルなライティング調整
- アニメーションの滑らかさ、リアルさの向上
